## Concentration

* 右键点击StoryBoard 可以查看所有的属性
* 如果是StoryBoard关联的字段要改名，command+点击 然后选 rename,这样关联才不会断
* 任何的 Instance value 必须初始化 （！可以不用）
* Instance value 可以设置 didSet 方法，用于这个value change 自动更新页面
